# discord-interactions

Handles Discord /commands.  

## Setup

Node version: [Node v18+](https://nodejs.org/en/download/)

**Requirements:**  
Copy `.env.defaults` as `.env`.  
Override the necessary keys.  

**Install dependencies:**  
`yarn`  

**Start application:**  
`yarn start`

## Endpoints

- **Get route**  
`GET /route`  
String  

- **Get subroute**  
`GET /route/subroute`  
String
